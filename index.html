Creating Interactive Bubbles with p5.js

Introduction:

In this tutorial, we will create an interactive animation using p5.js. The animation will feature bubbles that move around the canvas, change colors, and respond to user interactions.

Prerequisites:
Basic knowledge of JavaScript and p5.js is recommended.

Tutorial:

Step 1: Setting Up the Canvas**

let bubbles = [];

function setup() {
  createCanvas(400, 400);
}
  • We start by creating a canvas of 400x400 pixels and declaring an array to store the bubbles in for later.

Step 2: Creating the Bubbles

function createBubble() {
  return {
    x: random(width),
    y: random(height),
    radius: 0,
    color: color(random(255), random(255), random(255)),
    growSpeed: random(1, 5),
  };
}

function setup() {
  createCanvas(400, 400);

  for (let i = 0; i < 10; i++) {
    bubbles.push(createBubble());
  }
}
  • We’ve defined a function createBubble() that returns an object representing a bubble.
  • Each bubble has properties such as x, y, radius, color, and growSpeed.
  • In the setup() function, we create an array bubbles and populate it with 10 bubbles using the createBubble() function.

Step 3: Drawing and Animating the Bubbles

function draw() {
  background(82, 175, 199, 30);
  stroke(255);

  for (let i = 0; i < bubbles.length; i++) {
    let bubble = bubbles[i];
    bubble.radius += bubble.growSpeed;

    if (bubble.radius > 200) {
      bubble.radius = 0;
      bubble.x = random(width);
      bubble.y = random(height);
      bubble.color = color(random(255), random(255), random(255));
    }

    fill(bubble.color);
    ellipse(bubble.x, bubble.y, bubble.radius * 2);
  }
}
  • In the draw() function, we loop through each bubble in the bubbles array.
  • We update the radius of each bubble based on its growSpeed.
  • If a bubble’s radius exceeds 200, we reset it to its initial state with a new position and color.
  • We fill each bubble with its unique color and draw it as an ellipse with a diameter of radius * 2.

Step 4: Interactivity (Optional)

You can make the bubbles interactive by responding to user input. For example, you can change a bubble’s color when the mouse pointer is close to it or add more bubbles when the user clicks on the canvas.
